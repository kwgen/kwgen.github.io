管式输出

$ cat somefile.txt | pyhton somescript.py | sort

  1. cat somefile.txt 读取文件内容到标准输出sys.stdout
  2. pyhton somescript.py 读取文本逻辑处理然后输出
  3. sort 重标准输入sys.stdin读取所用文本，按字母排序，然后把结果写入标准输出
